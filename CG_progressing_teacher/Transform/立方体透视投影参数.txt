1、一点透视

VRP = [0 0 0]; VPN = [0 0 1]; VUP = [0 1 0];  % 指定观察平面和观察方向
PRP = [0.5 0.5 4];  % 投影参考点(投影中心或视点)

投影平面只与z轴相交

移动投影参考点PRP至(2,2,4)，可以观察到立方体更多的面

PRP = [2 2 4];  % 投影参考点(投影中心或视点)

2、二点透视

VRP = [0 0 0]; VPN = [1 0 1]; VUP = [0 1 0];  % 指定观察平面和观察方向
PRP = [0.5 2 4];  % 投影参考点(投影中心或视点)

投影平面与x轴和z轴相交


使VUP从y轴偏离45度，值为(1,1,0)，投影结果为立方体倾斜了

VUP = [1 1 0];
